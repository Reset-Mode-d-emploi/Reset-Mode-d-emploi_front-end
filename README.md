# Front-end for Reset Mode d'emploi

> Front-end of the project Reset Mode d'emploi 

## Install

### Prerequisites

If you have not already installed it, install [Node JS](https://nodejs.org/en/download/)

### Project

```
npm install
```

## Running

```
npm start
```

## Building

```
npm run build
```
